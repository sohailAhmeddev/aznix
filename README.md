# Ordway Labs Replica

This repository contains a Next.js application that replicates the home page layout and visual structure of Ordway Labs.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## Notes

- This app uses the Next.js App Router and Tailwind CSS.
- The page is a layout replica and uses simplified content and sections.
